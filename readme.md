## Hacker News Parser

This tool fetches Hacker News Articles from `https://hacker-news.firebaseio.com/v0/topstories.json` and displays it in the terminal.

### How to Run
1. Make sure rust is installed on your device
2. clone this repo
3. run `cargo run`

You will now see the top 10 Hacker News Stories!
